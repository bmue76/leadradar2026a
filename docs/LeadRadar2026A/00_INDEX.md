# LeadRadar2026A – Dokumentations-Index (Living)

## Was ist das?
Diese Doku ist die zentrale Referenz für alle Teilprojekte (Chats).  
Jedes Teilprojekt aktualisiert mindestens:
- Changelog / Entscheidungen (01_HANDBOOK.md)
- falls relevant: DB/API/Runbook Kapitel

## Inhalte
- 01_HANDBOOK.md – Architektur, Regeln, Prozesse, Projektprinzipien
- 02_DB.md – Datenbank, Migrationen, Seed-Strategie, Tenants/Users (ohne Secrets)
- 03_API.md – API Namespaces, Response Shapes, Error Codes, Auth/Tenant Scope
- 04_RUNBOOK.md – Lokales Setup, Deploy, Debugging, GoLive Checklisten
- 05_RELEASE_TESTS.md – Teststrategie, Smoke-Suites, Definition of Done
- _templates/ – Templates für Teilprojekte (Startprompt + Schlussrapport)
- _private/ – NICHT committen: echte Secrets, Kunden-IDs, interne Notizen (optional)
- _build/ – PDF Exporte (optional)
